# Uber Trip Analysis

## Project Overview
This project analyzes Uber trip data to understand travel patterns, trip purposes, categories, and overall ride behavior. The goal of this project is to clean, preprocess, and explore the dataset to identify meaningful insights that can support better decision-making.

This analysis demonstrates my ability to work with real-world datasets, perform data cleaning, apply preprocessing techniques, and use Python for exploratory data analysis.

## Objective
The main objectives of this project are:

- Clean and prepare Uber trip data for analysis
- Analyze trip categories and purposes
- Identify patterns in ride usage
- Perform feature encoding for categorical variables
- Build a structured and analysis-ready dataset
- Present insights in a clear and understandable way

## Dataset
The dataset used in this project is:

- `UberDataset.csv`

It contains Uber trip-related information such as trip category, purpose, and other ride details.

## Tools & Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook
- Data Cleaning
- Exploratory Data Analysis
- One-Hot Encoding

## Project Workflow
1. Imported the Uber trip dataset
2. Checked dataset structure and missing values
3. Cleaned and prepared the data
4. Encoded categorical columns such as `CATEGORY` and `PURPOSE`
5. Created a final processed dataset for analysis
6. Explored patterns and insights from the data
